## About this site

Built with [MkDocs](https://www.mkdocs.org/) and the
[Material](https://squidfunk.github.io/mkdocs-material/) theme; deploys
automatically to GitHub Pages on every push to `main`.

To add a page, drop a Markdown file under `docs/` and add it to the `nav:`
section of `mkdocs.yml`. See **[Contributing](guides/contributing.md)**.
